# Workflow

## Main workflow

1. Identify a naming gap, conflict, or drift.
2. Check existing approved terms.
3. Propose the minimal naming change.
4. Review impact across affected artifacts.
5. Accept, reject, or defer the change.
6. Update authoritative dictionary records.

## Workflow acceptance

A naming update is ready for review when:
- the affected concept is clearly defined
- collisions and forbidden synonyms are addressed
- impacted artifacts are listed
